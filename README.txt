Requirements:

1. Create a virtual environment
    python -m venv venv
    source venv/bin/activate  # On Windows: .\venv\Scripts\activate
2. Install:
- installed python
- installed selenium
- installed pytest
- installed webdriver-manager
- installed pytest-html
- installed html-testRunner

3. Run pytest or pytest --html=report.html
